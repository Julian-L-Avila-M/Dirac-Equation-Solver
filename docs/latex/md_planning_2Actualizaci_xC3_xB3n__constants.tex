Se realizaron modificaciones en la librería {\bfseries{Dirac Solver}} para simplificar el manejo de constantes y matrices de Pauli, eliminando la dependencia de la clase {\ttfamily Constants()}. La librería ahora\+:


\begin{DoxyItemize}
\item Funciona únicamente en {\bfseries{unidades naturales}}.
\item Permite acceder a constantes globales directamente desde el paquete.
\item Mantiene compatibilidad con las clases {\ttfamily Grid}, {\ttfamily Constant\+Spinor} y {\ttfamily Gaussian\+Packet}.
\item Incluye documentación Doxygen en {\ttfamily \doxylink{____init_____8py}{\+\_\+\+\_\+init\+\_\+\+\_\+.\+py}} y {\ttfamily \doxylink{initial__state_8py}{initial\+\_\+state.\+py}}.
\end{DoxyItemize}
