\doxysubsection{Descripción detallada}
Implementación concreta de un potencial de Coulomb regularizado. 

Esta clase deriva de la clase abstracta \doxylink{classPotential}{Potential} y define un potencial electrostático tipo Coulomb con una regularización en el origen (r = 0) para evitar singularidades numéricas.

La expresión implementada es\+:  \[
V(r) = -\frac{Z}{\|r\| + \epsilon}
\]

donde\+:
\begin{DoxyItemize}
\item $ Z $ es la carga nuclear o constante de acoplamiento.
\item $ \epsilon $ es un parámetro de regularización pequeño para estabilizar el cálculo.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Este potencial puede emplearse tanto en simulaciones 1D, 2D o 3D. 
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Si se usa un valor de {\ttfamily epsilon\+\_\+} demasiado pequeño, el integrador podría experimentar inestabilidad numérica en r≈0.
\end{DoxyWarning}
