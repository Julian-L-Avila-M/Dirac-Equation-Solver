\doxysubsection{Descripción detallada}
Clase base abstracta para algoritmos de evolución temporal (patrón Strategy). 

Define la interfaz genérica para los integradores temporales utilizados en el solver de la ecuación de \doxylink{namespaceDirac}{Dirac}. Cada subclase concreta implementa su propio esquema numérico de evolución temporal.

Este diseño permite desacoplar el método numérico del resto de la arquitectura del simulador.
