\chapter{Plan del Proyecto para un Solucionador de la Ecuación de Dirac}
\hypertarget{md_planning_2problems-to-solve}{}\label{md_planning_2problems-to-solve}\index{Plan del Proyecto para un Solucionador de la Ecuación de Dirac@{Plan del Proyecto para un Solucionador de la Ecuación de Dirac}}

Este documento describe el plan del proyecto para un solucionador de la ecuación de Dirac. El solucionador manejará la ecuación \$(i\textbackslash{}gamma\texorpdfstring{$^\wedge$}{\string^}\textbackslash{}mu \textbackslash{}partial\+\_\+\textbackslash{}mu -\/ m)\textbackslash{}psi=0\$ en unidades naturales, donde el bispinor \$\textbackslash{}psi \textbackslash{}in \textbackslash{}mathbb\{C\}\texorpdfstring{$^\wedge$}{\string^}4\$ es un vector columna de valores complejos. Las siguientes secciones detallan los problemas a abordar, junto con sus respectivos requisitos computacionales.
