\doxysubsection{Descripción detallada}
Implementación flexible de un potencial escalar definido por el usuario. 

Esta clase permite definir un potencial arbitrario a partir de una función escalar proporcionada por el usuario, ya sea desde C++ o desde Python (gracias a la interoperabilidad con {\bfseries{pybind11}}).

La función debe tener la forma\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{double}\ f(\textcolor{keyword}{const}\ std::vector<double>\&\ r);}

\end{DoxyCode}
