\doxysubsection{Descripción detallada}
Enlaces de C++ a Python usando Pybind11 para el núcleo de simulación \doxylink{namespaceDirac}{Dirac}. 

Este archivo define el módulo {\ttfamily \+\_\+core}, que expone las clases de C++ (potenciales, condiciones de borde, mallas y métodos de integración temporal) al entorno Python.

Las clases aquí expuestas se emplean desde Python para construir, configurar y ejecutar simulaciones de la ecuación de \doxylink{namespaceDirac}{Dirac} mediante un motor numérico en C++.

Estructura general del módulo\+:
\begin{DoxyItemize}
\item {\bfseries{\doxylink{classGrid}{Grid}\+:}} Representa la malla espacial discreta.
\item {\bfseries{\doxylink{classPotential}{Potential}\+:}} Jerarquía de potenciales físicos (Free, Scalar, Coulomb).
\item {\bfseries{\doxylink{classBoundaryCondition}{Boundary\+Condition}\+:}} Estrategias de frontera (Periódica, Absorbente).
\item {\bfseries{\doxylink{classTimeIntegrator}{Time\+Integrator}\+:}} Métodos de evolución temporal (FDTDLeapfrog).
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Este archivo debe compilarse como extensión dinámica (shared library) a través de CMake y Pybind11.
