\doxysubsection{Descripción detallada}
Contenedor de datos que agrupa todos los parámetros necesarios para una simulación del ecuación de \doxylink{namespaceDirac}{Dirac}. 

\begin{DoxyVerb}Clase de datos para contener todos los parámetros para una simulación de Dirac.
Este objeto es construido por el DiracProblemBuilder.
\end{DoxyVerb}


Esta clase no ejecuta la simulación directamente. Su propósito es almacenar de forma consistente todos los parámetros de entrada del problema\+: geometría, estado inicial, potencial, condiciones de borde y parámetros de tiempo.

\begin{DoxyNote}{Nota}
Este objeto es usualmente construido por Dirac\+Problem\+Builder. 
\end{DoxyNote}
