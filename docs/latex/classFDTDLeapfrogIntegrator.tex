\doxysubsection{Descripción detallada}
Implementación concreta de \doxylink{classTimeIntegrator}{Time\+Integrator} mediante el esquema FDTD de salto de rana (Leapfrog). 

Este integrador emplea el método {\itshape Finite-\/\+Difference Time-\/\+Domain (FDTD)} con un esquema temporal tipo "{}leapfrog"{}, ampliamente usado para ecuaciones de onda relativistas como la ecuación de \doxylink{namespaceDirac}{Dirac}.

El algoritmo alterna entre los estados ψ(t -\/ Δt), ψ(t) y ψ(t + Δt) para avanzar la solución sin necesidad de almacenar todas las configuraciones temporales.\hypertarget{classFDTDLeapfrogIntegrator_autotoc_md2}{}\doxysubsubsubsection{\texorpdfstring{Características principales}{Características principales}}\label{classFDTDLeapfrogIntegrator_autotoc_md2}

\begin{DoxyItemize}
\item Soporta dominios 1D, 2D y 3D.
\item Compatible con potenciales arbitrarios {\ttfamily \doxylink{classPotential}{Potential}}.
\item Integra condiciones de frontera mediante la clase {\ttfamily \doxylink{classBoundaryCondition}{Boundary\+Condition}}.
\item Expone la función de onda al entorno Python vía pybind11.
\end{DoxyItemize}\hypertarget{classFDTDLeapfrogIntegrator_autotoc_md3}{}\doxysubsubsubsection{\texorpdfstring{Referencias}{Referencias}}\label{classFDTDLeapfrogIntegrator_autotoc_md3}

\begin{DoxyItemize}
\item Taflove, A., \& Hagness, S. C. (2005). {\itshape Computational Electrodynamics\+: The Finite-\/\+Difference Time-\/\+Domain Method.}
\item Thaller, B. (1992). {\itshape The \doxylink{namespaceDirac}{Dirac} Equation.} 
\end{DoxyItemize}

Avanza un paso de tiempo completo según el esquema FDTD Leapfrog. 

Calcula el operador espacial y el término local del Hamiltoniano para cada punto de la malla, aplicando las matrices de \doxylink{namespaceDirac}{Dirac} correspondientes a cada dimensión. Luego, actualiza ψ(t+Δt) a partir de ψ(t) y ψ(t-\/Δt).

También aplica la condición de frontera definida por {\ttfamily boundary\+\_\+condition\+\_\+}.

\begin{DoxyNote}{Nota}
Actualmente implementa derivadas centrales de segundo orden y admite condiciones periódicas o absorbentes. 
\end{DoxyNote}
