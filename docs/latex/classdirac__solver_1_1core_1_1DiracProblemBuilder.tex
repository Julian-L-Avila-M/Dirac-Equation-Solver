\doxysubsection{Descripción detallada}
Implementación del patrón Builder para crear objetos Simulation\+Problem. 

Esta clase ofrece una interfaz fluida para configurar los componentes de una simulación del ecuación de \doxylink{namespaceDirac}{Dirac}. Facilita el proceso de validación y permite un flujo de construcción más legible.
