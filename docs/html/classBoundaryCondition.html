Descripción detallada

Clase base abstracta para condiciones de borde (patrón Strategy).

Esta clase define la interfaz para todas las estrategias de condiciones de borde. Permite que el solver principal se desacople del método específico utilizado para manejar los bordes de la malla de simulación.
