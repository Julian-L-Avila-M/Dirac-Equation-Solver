Descripción detallada

Implementación del patrón Builder para crear objetos SimulationProblem.

Esta clase ofrece una interfaz fluida para configurar los componentes de una simulación del ecuación de Dirac. Facilita el proceso de validación y permite un flujo de construcción más legible.

Ejemplo de uso:

builder = DiracProblemBuilder()
problem = (builder
.set_grid(my_grid)
.set_initial_state(my_packet)
.set_potential(V)
.set_boundary_condition("absorbing")
.set_time_parameters(dt, T)
.build())
